Data Pipeline and Acceleration

Raw CSVs land in Cloud Storage, BigQuery stores production analytics tables, and RAPIDS/cuDF can accelerate dataframe cleaning on NVIDIA GPUs. This Vercel version serves a precomputed dashboard for easy sharing; the Streamlit app in the repository keeps the interactive local and BigQuery workflow.
